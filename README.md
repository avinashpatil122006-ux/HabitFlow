# HabitFlow – Smart Habit Tracker

> **"Build Better Habits Every Day"**

HabitFlow is a professional-grade productivity application designed to help users establish consistent routines, maintain long-term streaks, and visualize their progress through advanced analytics and gamification. 

Built with a premium SaaS aesthetic, HabitFlow focuses on the psychology of consistency, rewarding users with XP and unlockable achievements.

## ✨ Key Features

- **Dynamic Dashboard:** Real-time progress monitoring with animated circular progress rings.
- **Visual Calendar:** Monthly overview with color-coded consistency heatmaps.
- **Advanced Analytics:** Detailed charts (Weekly, Category-wise) powered by Chart.js.
- **Gamified Experience:** Earn XP for every action and unlock professional badges.
- **SaaS Aesthetics:** Modern glassmorphism design with Dark/Light mode support.
- **Privacy First:** 100% client-side. All data stays in your browser's LocalStorage.
- **Data Portability:** Export and Import your data as JSON at any time.

## 🛠️ Technologies Used

- **HTML5:** Semantic architecture.
- **CSS3:** Custom properties (CSS Variables), Flexbox, Grid, and Glassmorphism effects.
- **Vanilla JavaScript (ES6+):** Modular logic without external frameworks.
- **LocalStorage API:** Persistent data storage.
- **Chart.js:** Robust data visualization.
- **Lucide Icons:** Clean, modern iconography.

## 🚀 Getting Started

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. No installation, server, or build step required!

## ⌨️ Keyboard Shortcuts

- `Ctrl + N`: Add New Habit
- `Ctrl + F`: Search Habits
- `Ctrl + D`: Toggle Dark Mode
